Projeto Final de Design de Software

  • Nome: Land of Myths
  • Curso: Engenharia de Computação
  • Período: Primeiro Semetre
  • Design : Sim
  • Escrita : Não
  • Descrição : Como projeto final desenvolvemos um game de sobrevivência por turnos, programado em Python usando o tkinter como framework e o firebase como banco de dados. O desafio era que o tkinter é um framework bem antigo e não foi criado para se fazer games,a magia foi fazer um game interessante e razoalvemente atraente com um framework desse. A programação foi pesada mas o resultado foi muito satisfatório.
